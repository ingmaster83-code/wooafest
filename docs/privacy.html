개인정보처리방침

최종 업데이트: 2026년 6월 29일

wooafest(이하 "서비스")는 회원가입 없이 이용 가능한 서비스로, 개인정보를 최소한으로 수집합니다.

1. 수집하는 정보

본 서비스는 다음과 같은 정보를 수집할 수 있습니다:

  • 위치 정보: 현재 위치 기반 주변 행사 검색 시 사용자의 동의를 받아 위도/경도를 일시적으로 처리합니다. 서버에 저장되지 않습니다.
  • 쿠키 및 사용 데이터: Google AdSense 광고 서비스 제공을 위해 쿠키를 사용합니다.
  • 자동 수집 정보: 서비스 이용 시 IP 주소, 브라우저 종류, 방문 일시 등이 자동으로 수집될 수 있습니다(Google Analytics).

2. 위치 정보 이용

현재 위치 기반 검색 기능 사용 시:

  • 브라우저의 Geolocation API를 통해 위도/경도를 수집합니다.
  • 수집된 위치 정보는 주변 행사 표시에만 사용되며 서버에 저장되지 않습니다.
  • 위치 정보 제공을 거부할 경우 해당 기능 이용이 제한될 수 있습니다.

3. 광고 서비스

본 서비스는 Google AdSense를 통한 맞춤형 광고를 제공합니다. Google은 쿠키를 사용하여 사용자의 이전 방문 기록을 기반으로 광고를 게재할 수 있습니다.

  • Google 개인정보처리방침: https://policies.google.com/privacy
  • 광고 맞춤 설정 해제: https://www.google.com/settings/ads

4. 제3자 데이터 제공

본 서비스는 다음 공공 데이터를 활용합니다:

  • 한국문화정보원 한눈에보는문화정보 API
  • 전국문화축제표준데이터 (공공데이터포털)
  • 전국공연행사정보표준데이터 (공공데이터포털)

개인정보는 제3자에게 제공되지 않습니다.

5. 카카오맵 API

지도 서비스 제공을 위해 카카오맵 API를 사용합니다. 카카오의 개인정보처리방침은 카카오 개인정보처리방침을 참조하세요.

6. 개인정보 보유 및 파기

본 서비스는 서버를 운영하지 않으며 사용자 데이터를 저장하지 않습니다. 모든 기능은 브라우저 내에서 처리됩니다.

7. 이용자 권리

이용자는 언제든지 브라우저 설정에서 쿠키를 삭제하거나 위치 정보 제공을 거부할 수 있습니다.
