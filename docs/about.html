🎪 우아축제 소개

전국 축제·공연·행사를 한 곳에서 — 완전 무료

서비스 소개

우아축제는 전국의 축제, 공연, 전시, 문화행사 정보를 한눈에 볼 수 있는 무료 서비스입니다. 한국문화정보원 공공 API와 전국 문화축제·공연행사 표준 데이터를 활용하여 매일 자동으로 정보를 갱신합니다.

주요 기능

  • 🗓️ 이번 주말 행사 — 지금 당장 갈 수 있는 행사 빠른 검색
  • 🆓 무료행사 필터 — 비용 걱정 없이 즐길 수 있는 행사 모아보기
  • 👶 아이랑 행사 — 가족이 함께 즐길 수 있는 행사 필터
  • 📍 지역별 검색 — 전국 17개 시도별 행사 목록
  • 🗺️ 카카오맵 연동 — 행사 위치 지도 확인 및 길찾기
  • 📅 문화캘린더 — 월별 행사 일정 한눈에 보기
  • ⏰ D-Day 카운트다운 — 마감 임박 행사 강조 표시

데이터 출처

  • 한국문화정보원 한눈에보는문화정보조회서비스 API
  • 전국문화축제표준데이터 (공공데이터포털)
  • 전국공연행사정보표준데이터 (공공데이터포털)
